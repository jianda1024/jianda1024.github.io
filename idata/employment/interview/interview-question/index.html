    面试问题

    面试问题 #

    记住：没有任何面试官会和你闲聊天 #

    1、你有什么兴趣爱好？ #

    • 避免太闲或太险的爱好
    • 要给面试官正面的联想

    2、你为什么离开上家公司？ #

    • 求发展、不可抗力

    3、你认为自己最大的缺点是什么？ #

    • 不要与职位冲突
    • 不要是致命的缺点
    • 确实是缺点，但对工作影响不大

    4、介绍下你的实习经历/项目经验？ #

    • 突出重点、注意细节

    5、你如何评价自己的大学生活？ #

    • 学到了什么、会做什么

    6、与主管意见有分歧，你会怎么做？ #

    • 首先服从、事后请教

    7、你有什么样的职业规划？ #

    • 不与职位冲突的前提下，如实回答
    • 规划最好与公司的发展相契合
    • 强调做好本职工作

    8、为什么选择我们公司？ #

    • 对行业的了解
    • 对公司的认同
    • 对职业的热爱
    • 有信心做好这份工作
