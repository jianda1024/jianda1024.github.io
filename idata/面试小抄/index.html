	先是在中软国际做华为项目的开发。使用的是OWS集成开发工具和Springcloud技术栈，负责对基站告警处理平台进行开发升级。

    项目难点 #

    • IPD项目
    问题：
业务接口调用超时，优化性能

解决：
1、使用IDEA插件arthas，获取接口各个语句的耗时情况，找到耗时较多的代码
2、在数据库性能方面，尝试优化，比如
	2.1、优化查询语句：避免使用不必要的子查询和复杂的函数；确保查询条件使用了合适的索引。
	2.2、建立和优化索引：根据经常用于查询、连接和排序的字段创建索引。
	2.3、调整数据库配置：合理配置内存缓冲区、连接数、线程池等参数。
	2.4、分库分表：当数据量过大时，将表按照一定规则进行水平或垂直分表，或者进行数据库的垂直分割和水平分割功能。
3、在代码逻辑上，尝试优化，比如
	3.1、对于没有前后依赖的逻辑，进行并行处理
	3.2、对于循环递归类的逻辑，看能否进行优化缩减
	3.3、相似的逻辑调用，尝试进行合并统一调用
4、将复杂接口，拆分为几个简单接口
5、对数据进行缓存处理
6、网上查询相关解决方案，或向同事进行求助
7、最后实在解决不了，向领导反馈
8、有优化方案后，如果改动复杂，则进行方案评估
示例：批量计算项目某个指标的接口，

问题：Jalor框架内的excel导出

解决：

    • 生命周期项目
    问题：
项目分为多个微服务，每个微服务都有多个质量衡量指标，每个指标都有各自的阈值，每次代码提交，都可能会影响这些指标。
需要每天实时监控这些指标，当天内不允许有指标超过阈值。质量指标只能一个服务一个服务，在监控平台上人工查看，耗时耗力

解决：
使用自动化脚本，代替人工监控
1、各指标是通过监控平台网页读取，选择使用python作为脚本语言，查询python资料，编写功能
2、模拟账号登陆监控平台，并抓取各微服务的质量指标
3、处理各指标，对比各自阈值，生成告警信息
4、生成告警邮件，发送到监控人邮箱
5、使用windows自带定时任务功能，定时运行脚本

    产品数字化平台 - 生命周期 #

    • 项目描述：产品数字化平台，支撑产品、版本、单板的生命周期规划管理流程，发布产品、版本、单板的生命周期信息查询；增加产品组合相关流程能力，产业销售目录维护、路标管理以及组合策略管理等功能特性；
    • 技术相关：Jalor、Spring、SpringMVC、TKMybatis、PgSQL、Redis等
    • 职责描述：每月两个版本的平台迭代开发任务、开源软件可信维护、安全扫描问题整改、代码质量指标看护
    • 项目收获：积累了项目开发经验，加深了对软件安全方面的理解。

    智慧城市大数据平台 - 数据集成 #

    • 项目描述：数据集成，是一个云化的ETL平台。ETL是将业务系统的数据经过抽取、清洗转换之后加载到数据仓库的过程，目的是将企业中的分散、零乱、标准不统一的数据整合到一起，为企业的决策提供分析依据。
    • 技术相关：Kettle、Spring、SpringMVC、Mybatis-plus、PostgreSQL、Redis、xxl-job等。
    • 职责描述：将Kettle已经有的本地算子，转换以插件形式加载到数据集成平台。并负责开发一些定制化的算子，及部分数据集成平台独有的功能。
    • 项目收获：学习了Kettle这个ETL开源工具，掌握了其算子的开发。积累了ETL的经验。

    深圳发改委数据清洗 #

    • 项目描述：发改委有很多统计数据来源为文件，需要解析这些文件导入数据库，为后续数仓数据清洗做准备。
    • 技术相关：Spring、SpringMVC、Mybatis-plus、PostgreSQL、达梦、Redis、POI、OCR。
    • 职责描述：负责模板配置、Excel解析导入、PDF解析导入、历史列表、数据预览功 能的开发。
    • 项目收获：熟悉了POI组件，了解了OCR技术。能够适应客户频繁变动的需求，快速响应得出结果。锻炼了自己对于具体的业务开发能力。

    数据治理元数据采集 #

    • 项目描述：大数据平台，数据来源自不同类型的数据库，采集各种数据库的元数据，才能进行数据库的操作。
    • 技术相关：Spring、SpringMVC、Mybatis-plus、PostgrepSQL、Oracle、达梦、MySQL、SQL Server、DB2、Greenplum、HBase、Hive、人大金仓、Gbase、HDFS。
    • 职责描述：负责项目环境的搭建及项目功能的开发。采集的元数据包括库信息、表信息、列信息、索引信息、触发器信息、外键信息、主键信息、文件信息（HDFS）等。
    • 项目收获：进一步了解各种类型的数据库。加深了对 sql 基础的学习与掌握。

    华为 OWS 平台告警处理系统 #

    • 项目描述：OWS是一个开放的、可编程的软件驱动平台，具有微服务架构，运用DevOps模式，能够简单快速的创建新的应用程序。华为电信基站每天上报大量故障信息，告警处理系统，可过滤分析这些信息，并分发派单给一线人员，使问题得到快捷高效的解决。 所负责的微服务有：活动告警服务、历史告警、告警规则服务等。
    • 技术相关：使用的工具/技术有OWS、JavaScript、shell、SVN、Git、MobaXterm、XMind等
    • 职责描述：一线问题单处理、月度版本出包验证、平台功能需求开发。
    • 项目收获：对于陌生的工具与业务，我能较快速的上手，较好的完成工作任务，得到华为方面的认可，绩效较好，被提为骨干员工。
