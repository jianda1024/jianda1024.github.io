    简历编写 #

    编写原则 #

    • 简历编写的原则：趋利避害
    • 简历要好看：形式上的好看、内容上的好看
    • 简历编写的时机：时不时更新下简历，总结检讨自己的成长
    • HR 如何刷选简历
      • 是快速浏览，而不是仔细阅读
      • 是寻找关键字，而非逐句分析
    • HR 最讨厌的简历
      • 又臭又长—–简单有力
      • 千篇一律—–与众不同
      • 含糊不清—–交代具体

    注意事项 #

    基本资料 #

    • 必不可少的选项，不欲盖弥彰
      • 如姓名、性别、出生年月、年龄、婚姻状况、联系方式（手机号、邮箱）
    • 可有可无的选项，不自曝其短
      • 如籍贯、民族、照片、显居住地、英文水平

    求职意向 #

    • 意向职位、意向城市：做痴心小种子，别做花心大萝卜
    • 所属行业：不建议自设门槛
    • 期望薪资

    教育经历 #

    • 从高中开始倒序来写
    • 学会了什么，而不是学了什么

    个人技能 #

    • 归纳清晰、分清主次、合理行文

      反例：会Linux常用命令，能在Linux上搭建LAMP环境
正例：掌握Linux日常管理操作，熟悉Linux服务器的LAMP环境部署、优化与安全

    工作经历 #

    • 是怎么做好这份工作的，而不是这份工作是做什么的

    项目经历 #

    • 基本原则：who、when、where、how、why、what
    • 基本标准：真、实、扩展

    自我评价 #

    • 突出与众不同的能力或经历

    简历投递 #

    • 胆大：撑死胆大的，饿死胆小的
    • 心细：心中有猛虎，细能嗅蔷薇
    • 脸皮厚：脸皮薄吃不着，脸皮厚吃个够
    • 纸质简历的小细节
      • 一定要带简历
      • 简历要干净整洁
      • 双手递递简历，文字朝向对方，方便对方直接查看
